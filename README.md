# Discord-Bot-Control
A open source python code to control your Discord Bot's DM's and Messages in servers

To see your messages in DM's refresh the DM Page (Click "Open DM by id" again)

# How to set up?

Open your cmd and type
Git clone https://github.com/cleyx1343-a11y/Discord-Bot-Control
cd Discord-Bot-Control
pip install -r requirements.txt

# How to connect to bot?
Copy your bot's token by Discord Dev Portal, then open the .py file and paste it into where it says BOT_TOKEN="Your_Bot_Token_here"

# Start the app
py main.py
